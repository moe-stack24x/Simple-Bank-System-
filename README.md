# 🏦 Simple Bank System

A **console-based banking system** built with C++ to simulate basic bank operations and practice programming logic.

## 🌟 Features

* Add new clients
* Delete or update client information
* Search for clients by account number
* Deposit and withdraw money
* View total balances

## 🛠️ Technologies Used

* **Language:** C++
* **Concepts:** Functions, Structs, Vectors, File Handling, Menu-driven system
